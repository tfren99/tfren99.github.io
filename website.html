project overview

  • Personal project
  • Originally started in high school, when I was mildly interested in a career in web design. Since then I have continued working on the site on and off as a way to practice my "eye for design".

article

This website is a project that I have been working on for many years. I have overhauled it several times as my design tastes change with time. With this iteration, I've challenged myself to code the entire website myself, without using any public templates or copy-pasting from any online forums or GPT chatbots. The site is hosted through Github pages, and the source code can be found on my Github profile.
